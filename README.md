# AM-stepbystep

Short and sweet step-by-step lab walkthrough of **Azure Migrate**.

## Lab Guide

The full lab guide is in **[lab-guide.md](lab-guide.md)**.

It covers:
1. Creating an Azure Migrate project
2. Enabling preview features via the feature-flag URL
3. Importing an offline assessment ZIP file (included in this repo)
4. Reviewing the Action Center for discovery issues
5. Exploring the discovered inventory (servers, databases, web apps)
6. Tagging workloads (including special `AzM_Environment` and `AzM_MigrationIntent` tags)
7. Creating application groupings (manually and via CSV)
8. Creating an assessment
9. Reviewing assessment reports and cost estimates
10. Using the Azure Migrate AI Agent *(coming soon)*
